# Tetris
Tetris game for Object Oriented project

## requirements
- smfl

## compilation
g++ *.cpp -lsfml-graphics -lsfml-window -lsfml-system

## Controls
|   |   |
|---|---|
| ←,→  |  move piece horizontally |
| ↑, x | rotate piece clockwise |
| z | rotate piece counterclockwise |
| ↓ | speed up falling |
| p, esc | pause |
